# Gestão de Frotas - PAV

Aplicativo mobile desenvolvido para cadastrar e inativar veículos da frota, criar relatório de gastos com carro e combústivel e registrar abastecimento. Aba gerência para extração dos dados em XLS.

## Tecnologias Utilizadas

- **React Native**
- **Nodejs** (API)
- **Prisma** (ORM)
- **Postgree** (DB)
- **Vercel** (Hospedagem e Serverless)
